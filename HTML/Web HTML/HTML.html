Pengertian HTML

Apa itu HTML? HTML atau HyperText Markup Language adalah bahasa yang menggunakkan markah atau penanda untuk membuat web.
Penanda itu apa? Penanda atau Tag itu contohnya seperti < head >, < body >, < h1 >, < p > HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web.

Bila diibaratkan, HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar.


((Gambar 1.1) Penjelasan singkat mengenai HTML, CSS, dan Javascript)

Dalam membuat halaman web, HTML tidak sendirian. Ada bahasa lain lagi yang menjadi pelengkapnya, yakni CSS dan Javascript.
CSS atau Cascading Style Sheets berfungsi untuk memperindah tampilan web. Sedangkan Javascript berfungsi untuk menambah efek atau fungsi lain ke dalam web agar web terlihat semakin hidup. Jadi, urutan untuk yang akan kita pelajari adalah...

  1. HTML atau HyperText Markup Language
  2. CSS atau Cascading Style Sheets
  3. Javascript

Sejarah HTML

HTML awalnya muncul pada tahun 1980. Seorang Ilmuwan bernama Tim Berners-Lee bekerja di CERN. CERN bukanlah perusahaan teknologi ataupun internet
melainkan Conseil Européen pour la Recherche Nucléaire. Yaitu Komisi Eropa untuk Penelitian Fisika Nuklir. Para peneliti membutuhkan suatu cara agar bisa saling berbagi dokumen dan hasil riset mereka.
Sehingga, Tim membuat ENQUIRE, software hypertext yang akan digunakan oleh CERN untuk berbagi dokumen. Lalu, Tim mengembangkan hypertext nya pada tahun 1989 yang berbasis internet bersama dengan Robert Cailliau. Namun, proyek ini tidak resmi digunakan oleh CERN.

Karena Tim pantang menyerah, pada akhir tahun 1991, Tim Berners-Lee merilis sebuah dokumen yang berjudul : "HTML Tags". Dokumen ini berisikan 18 tags awal yang menjadi konsep dasar HTML.
Berdasarkan hasil dokumen milik Tim,HTML sebenarnya dirancang berdasarkan pada konsep bahasa markup yang dikenal dengan SGML (Standard Generalized Markup Language). SGML itu sendiri adalah standar internasional untuk membuat dokumen dengan tanda (markup) seperti paragraf, list, heading, dan lain-lain.
Yang berarti, HTML adlaah implementasi dari SGML. Tapi, tidak semua tag yang ada di HTML itu berasal dari SGML. Hyperlink, adalah salah satu hasil dari penemuan Tim Berners-Lee. Segera, ide mengenai HTML ini disebarkan melalui internet dan menjadi perhatian baru bagi para ilmuwan saat itu.

Urutan Perkembangan dan Versi HTML

HTML selama perjalanannya terus mengalami perubahan dan perkembangan. Berikut adalah urutan perkembangannya

  1. [Draft] HTML 1.0 (Juni 1993),versi HTML pertama, namun tidak resmi dirilis
  2. HTML 2.0 (24 November 1995),versi HTML kedua yang resmi pertama kali beredar di pasaran. DIrilis oleh IETF atau Internet Engineering Task Force
  3. [Draft] HTML 3.0 (28 Maret 1995), versi yang sayangnya gagal beredar, karena banyak perubahan yang memicu perdebatan
  4. HTML 3.2 (14 Januari 1997),versi resmi yang dirilis W3C atau World Wide Web Consortium pertama kali
  5. HTML 4.0 (24 April 1998),versi pengembangan dari yang sebelumnya
  6. HTML 4.01 (24 Desember 1999),versi perbaikan dari HTML 4.0
  7. XHTML 1.0 (26 Januari 2000),versi pengembangan dari HTML 4.01 dengan mengadopsi XML
  8. XHTML 2.0 (Augustus 2002—Juli 2006),versi kedua dari XHML
  9. HTML 5.0 (28 Oktober 2014),versi HTML saat ini
Versi draft adalah versi yang tidak pernah resmi rilis di pasaran. Bentuknya hanya draft spesifikasi dan tidak pernah digunakan resmi.
Versi yang akan kita gunakan adalah Versi HTML 5.0.
